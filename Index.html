User Guide

Navigating the Application

Welcome to our Event Calendar application! Here's a guide on how to use it:

  • Home Page: You can start by visiting the home page, where you'll find information about the application and a "Get Started" button.
  • Login: Click "Get Started" to log in using your Google account. This will redirect you to Google's authentication page.
  • Google Authentication: On Google's page, log in with your Google credentials and grant necessary permissions to access your Google Calendar.
  • Dashboard: After successful authentication, you'll be redirected to the dashboard. Here, you can:
    • Create a new calendar event by filling out the form and clicking "Create Event."
    • View and manage your existing calendar events.
    • Log out when you're done using the application.

How the UI Interacts with the Backend and Google API

Our application interacts with the backend and Google API to create calendar events as follows:

  1. When you click "Create Event" on the dashboard, the UI collects event details (event summary and start time).
  2. The UI sends these details to the backend via an HTTP POST request to the `/create-event` route.
  3. The backend uses Google API credentials and the stored authentication token to make an authenticated API request to create a calendar event on your Google Calendar.
  4. Once the event is created successfully, the backend sends a response to the UI confirming the event's creation.

That's it! You can now start using our application to manage your calendar events seamlessly.
